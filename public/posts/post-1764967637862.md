# Win11 安装跳过联网及永久激活指南

Win11 终于加入了 macOS 上早就有的“文件管理器多标签页”功能，这确实是个痛点。不过为了这个功能，不得不忍受 Win11 那反人类的“右键 > 显示更多选项”。

这里记录一下重装系统时的两个关键步骤：**跳过强制联网**和**系统激活**，顺便解决一下右键菜单的问题。

## 1. 安装时跳过强制联网

在安装向导走到连接网络（Wi-Fi）的界面时，微软默认隐藏了“我没有 Internet 连接”的按钮。

1.  拔掉网线（确保未连接网络）。
2.  按下 **`Shift + F10`** 调出命令提示符（部分笔记本可能需要按 `Shift + Fn + F10`）。
3.  输入以下命令并回车，系统会自动重启：

```cmd
OOBE\BYPASSNRO
````

> **说明：** 重启后再次进入网络界面，右下角会出现“**我没有 Internet 连接**”的选项，点击它并选择“**继续执行受限设置**”即可建立本地账户进入桌面。

## 2\. 系统激活 (MAS)

使用开源且广泛好评的 [Microsoft Activation Scripts (MAS)](https://massgrave.dev/) 进行激活。

1.  进入系统后，右键开始菜单，选择 **PowerShell (管理员)** 或 **终端 (管理员)**。
2.  输入以下命令并回车：

<!-- end list -->

```powershell
irm [https://massgrave.dev/get](https://massgrave.dev/get) | iex
```

3.  在弹出的菜单窗口中，输入数字 **`1`** (HWID Activation)。
      * *HWID 是数字权利激活，绑定主板，重装系统联网后会自动激活，属于永久激活。*

## 3\. 附：恢复 Win10 右键菜单

在终端（管理员）中执行：

```powershell
reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
```

**恢复 Win11 默认菜单：**

```powershell
reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f
```

> **注意：** 执行完命令后，需要**重启资源管理器**（打开任务管理器 -\> 找到 Windows 资源管理器 -\> 右键重启）或重启电脑生效。
